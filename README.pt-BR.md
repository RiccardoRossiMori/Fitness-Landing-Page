# 🏋️‍♂️ Fitness Studio Platform
**Landing Page & Scalable Digital Solution for Gyms and Fitness Studios**

Site disponível no link: [https://fitness-landing-page-wine.vercel.app](https://fitness-landing-page-wine.vercel.app
)


---~~~~

### Visão Geral
Fitness Studio Platform é uma **solução web moderna e focada em crescimento**, desenvolvida para academias e estúdios fitness que desejam **atrair novos alunos**, melhorar a comunicação e construir uma **presença digital profissional e escalável**.

O projeto começa como uma **landing page de alta conversão**, mas já é estruturado para evoluir para uma **aplicação web completa**.

---

### Valor para o Negócio
- Aumento de matrículas
- Comunicação direta via e-mail
- Centralização de novidades e promoções
- Base sólida para funcionalidades futuras
- Imagem profissional e confiável

---

### Funcionalidades Principais
- Landing page moderna e responsiva
- Apresentação da academia e serviços
- Horários de funcionamento
- Área de contato
- Inscrição na newsletter
- Captação de novos interessados

A newsletter permite informar:
- Feriados e fechamentos
- Promoções
- Novas aulas
- Atualizações de equipamentos e serviços

---

### Funcionalidades Futuras (Roadmap)
- Sistema de login com níveis de acesso:
    - Usuário
    - Staff
    - Administrador
- Acesso do staff via código exclusivo
- Sistema de gerenciamento de conteúdo:
    - Notícias
    - Eventos
    - Artigos informativos
    - Fotos e vídeos
- Conteúdo dinâmico na página inicial
- Integração com QR Code

---

### Tecnologias Utilizadas

*Frontend*
- React (Vite)
- HTML5 / CSS3
- Bootstrap 5

*Form Handling*
- EmailJS (envio de e-mails a partir do formulário de contato)

*Deploy*
- Vercel

### Justificativa Técnica
O stack foi escolhido para manter o projeto simples, performático e alinhado com a natureza de uma landing page focada em conversão, evitando complexidades desnecessárias como banco de dados ou sistemas de autenticação.
